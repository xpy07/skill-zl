# 工作空间说明

本目录是用户工作空间，用于存储检索结果、风格指南、撰写的文档和校对报告。

## 目录结构

```
workspace/
├── search-results/               # 检索结果存储
│   └── {关键词}-{日期}.json      # 检索结果文件
├── style-guides/                 # 风格指南存储
│   └── {领域}-style-guide.md     # 风格指南文件
├── documents/                    # 撰写的文档
│   └── {文档名称}.docx           # 撰写的文档
└── proofread-reports/            # 校对报告
    └── proofread-{文档名称}-{日期}.md  # 校对报告
```

## 文件命名规范

### 检索结果

- 格式：`{关键词}-{日期}.json`
- 示例：`锂电池正极材料-2024-01-15.json`

### 风格指南

- 格式：`{领域}-style-guide.md`
- 示例：`专利申请文件-style-guide.md`

### 撰写的文档

- 格式：`{文档名称}.docx`
- 示例：`锂电池正极材料专利申请.docx`

### 校对报告

- 格式：`proofread-{文档名称}-{日期}.md`
- 示例：`proofread-锂电池正极材料专利申请-2024-01-15.md`

## 使用说明

1. **检索结果**：检索模块会自动将检索结果保存到 `search-results/` 目录
2. **风格指南**：学习模块会自动将风格指南保存到 `style-guides/` 目录
3. **撰写的文档**：撰写模块会自动将文档保存到 `documents/` 目录
4. **校对报告**：校对模块会自动将校对报告保存到 `proofread-reports/` 目录

## 注意事项

1. 请勿手动删除或修改工作空间中的文件
2. 如果需要清理空间，请备份重要文件后再删除
3. 工作空间中的文件仅在当前会话中有效
