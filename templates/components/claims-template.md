# 权利要求书模板

## 基本格式

```markdown
1. 一种[技术领域]，其特征在于，包括：
   [技术特征1]；
   [技术特征2]；
   [技术特征3]。

2. 根据权利要求1所述的[技术领域]，其特征在于，所述[技术特征1]包括：
   [进一步限定1]；
   [进一步限定2]。

3. 根据权利要求1或2所述的[技术领域]，其特征在于，所述[技术特征2]包括：
   [进一步限定3]。

4. 根据权利要求1-3中任一项所述的[技术领域]，其特征在于，所述[技术特征3]包括：
   [进一步限定4]。

[...继续从属权利要求...]
```

## 撰写规则

### 独立权利要求

**格式**：
```
1. 一种[技术领域]，其特征在于，包括：
   [技术特征1]；
   [技术特征2]；
   [技术特征3]。
```

**要求**：
- 包含解决技术问题的全部必要技术特征
- 前序部分：技术领域 + 引用部分（如果有）
- 特征部分：使用"其特征在于"引出
- 技术特征：用分号分隔，最后一个用句号

### 从属权利要求

**格式**：
```
2. 根据权利要求1所述的[技术领域]，其特征在于，所述[技术特征1]包括：
   [进一步限定1]；
   [进一步限定2]。
```

**要求**：
- 引用在前的权利要求
- 对引用的权利要求的进一步限定
- 引用关系清晰正确

## 引用规则

| 引用方式 | 说明 | 示例 |
|----------|------|------|
| 单一引用 | 引用一项权利要求 | 根据权利要求1所述的... |
| 多项引用 | 引用多项权利要求 | 根据权利要求1-3中任一项所述的... |
| 选择引用 | 引用多项中的任一项 | 根据权利要求1或2所述的... |

## 术语规则

1. **一致性**：全文使用相同术语
2. **准确性**：使用正确的专业术语
3. **明确性**：术语含义明确，无歧义

## 常见错误

| 错误类型 | 示例 | 正确做法 |
|----------|------|----------|
| 引用错误 | 根据权利要求1和2所述的... | 根据权利要求1或2所述的... |
| 术语不一致 | 处理器...处理单元... | 全文统一使用"处理器" |
| 特征遗漏 | 缺少必要技术特征 | 确保包含全部必要技术特征 |
| 格式错误 | 缺少"其特征在于" | 使用标准格式 |
