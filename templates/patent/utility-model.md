# 实用新型专利申请文件模板

## 文件结构

```
[实用新型名称]

一、技术领域
[本实用新型所属的技术领域]

二、背景技术
[现有技术的描述，包括相关专利引用]
[现有技术的缺点和不足]

三、发明内容
1. 发明目的
[本实用新型要解决的技术问题]

2. 技术方案
[本实用新型的技术方案详细描述]

3. 有益效果
[本实用新型的技术效果和优势]

四、附图说明
[附图的详细说明，包括各部件标记]

五、具体实施方式
[详细描述如何实施该实用新型，包含多个实施例]

六、权利要求书
[独立权利要求 + 从权权利要求]

七、说明书摘要
[不超过300字的摘要]

八、摘要附图
[摘要附图]
```

## 与发明专利的区别

| 项目 | 发明专利 | 实用新型 |
|------|----------|----------|
| 保护对象 | 方法/产品 | 产品（形状/结构） |
| 创造性要求 | 较高 | 较低 |
| 审查方式 | 实质审查 | 初步审查 |
| 保护期限 | 20年 | 10年 |
| 附图要求 | 可选 | 必须有附图 |

## 撰写要点

### 技术领域
- 简要说明本实用新型所属的技术领域
- 通常1-2句话

### 背景技术
- 描述现有技术
- 引用相关专利
- 分析现有技术的缺点
- 说明本实用新型要解决的问题

### 发明内容
- **发明目的**：要解决的技术问题
- **技术方案**：详细的技术方案（必须有形状/结构特征）
- **有益效果**：技术效果和优势

### 附图说明
- 必须有附图
- 说明各附图的内容
- 标注各部件的标记
- 说明各部件的功能

### 具体实施方式
- 详细描述实施步骤
- 包含多个实施例
- 引用附图标记
- 确保支持所有权利要求

### 权利要求书
- **独立权利要求**：包含形状/结构特征
- **从属权利要求**：对独立权利要求的进一步限定
- 引用关系清晰
- 术语一致

### 说明书摘要
- 不超过300字
- 概括发明内容
- 不包含权利要求

### 摘要附图
- 选择最能代表技术方案的附图
- 作为摘要附图

## 格式规范

| 项目 | 规范 |
|------|------|
| 标题 | 黑体，小二号 |
| 正文 | 宋体，小四号 |
| 行距 | 1.5倍行距 |
| 页边距 | 上下2.54cm，左右3.17cm |
| 编号 | 一、二、三...（一级）1、2、3...（二级）(1)、(2)、(3)...（三级） |
| 附图 | 必须有附图，附图标记连续编号 |
