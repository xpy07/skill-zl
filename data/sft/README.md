# 蒸馏数据说明

## 数据来源

基于以下三件专利公开文件构建的训练数据：

| 专利号 | 名称 | 技术领域 |
|--------|------|---------|
| CN202511361974 | 一种用于弃渣场边坡的滑面失稳预测方法 | 岩土工程 |
| CN202511551341 | 基于多无人机协同的运动物体三维模型重构系统和方法 | 计算机视觉/无人机 |
| CN202511627186 | 一种基于FPGA的低延时高位宽乘法器的实现方法 | 数字电路设计 |

## 训练数据分布

| 任务类型 | 文件 | 数量 | 说明 |
|---------|------|------|------|
| 创造性评估 | creativity_*.jsonl | 3 | 三步法结构化分析 |
| 文件撰写 | drafting_001.jsonl | 1 | 完整专利申请文件（权利要求书+说明书+摘要） |
| 审查意见答复 | office_action_001.jsonl | 1 | 争辩+修改策略 |
| 客户咨询 | consultation_001.jsonl | 1 | 流程/费用/策略指引 |

## 数据格式

标准 JSONL 格式，每条数据包含 messages 数组：
```json
{
  "messages": [
    {"role": "system", "content": "系统提示词"},
    {"role": "user", "content": "用户输入"},
    {"role": "assistant", "content": "期望输出"}
  ]
}
```

## 后续扩展

2. 用 Teacher 模型（Qwen3-235B）批量生成更多样本
3. 增加不同难度等级的测试案例
