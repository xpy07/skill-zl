# 检索模块

## 功能说明

多源检索专利、论文、百科、法律条文，为撰写提供信息支撑。

## 触发词

"检索"、"搜索"、"查找"、"查询"、"找一下"

## 数据源

### 专利数据库（侧重国内）

| 数据源 | 类型 | 优先级 | 说明 |
|--------|------|--------|------|
| CNIPA | 中国专利 | 高 | 国家知识产权局，中国专利检索 |
| Google Patents | 全球专利 | 高 | 全球专利检索，支持中英文 |
| USPTO | 美国专利 | 中 | 美国专利商标局 |
| EPO | 欧洲专利 | 中 | 欧洲专利局 |
| WIPO | PCT国际申请 | 中 | 世界知识产权组织 |

### 学术论文

| 数据源 | 类型 | 优先级 | 说明 |
|--------|------|--------|------|
| 知网 (CNKI) | 中文期刊/学位论文 | 高 | 中国最大的学术论文数据库 |
| 万方 | 中文期刊/会议论文 | 中 | 中国学术论文数据库 |
| Semantic Scholar | 英文论文 | 中 | AI驱动的学术搜索（免费API） |
| arXiv | 预印本论文 | 中 | 预印本论文（免费API） |
| PubMed | 医学/生物论文 | 低 | 医学论文数据库（免费API） |

### 百科知识

| 数据源 | 类型 | 优先级 | 说明 |
|--------|------|--------|------|
| 百度百科 | 中文百科 | 中 | 中文百科知识 |
| 维基百科 | 多语言百科 | 低 | 多语言百科知识（免费API） |

### 法律条文

| 数据源 | 类型 | 优先级 | 说明 |
|--------|------|--------|------|
| 专利法 | 法律 | 高 | 中华人民共和国专利法 |
| 专利法实施细则 | 法规 | 高 | 专利法实施细则 |
| 审查指南 | 规范 | 高 | 专利审查指南 |

## 检索流程

```
用户输入：关键词/技术方案
    ↓
[步骤1] 提取关键词
  - 从用户输入中提取核心技术关键词
  - 生成同义词、近义词扩展
  - 确定检索策略
    ↓
[步骤2] 专利检索（最高优先级）
  - 检索CNIPA（中国专利）
  - 检索Google Patents（全球专利）
  - 筛选相关专利（按相关度排序）
  - 提取专利摘要、权利要求、说明书
    ↓
[步骤3] 论文检索
  - 检索知网、万方（中文论文）
  - 检索Semantic Scholar、arXiv（英文论文）
  - 筛选相关论文
  - 提取论文摘要、结论
    ↓
[步骤4] 百科检索
  - 检索百度百科、维基百科
  - 提取基础概念、技术背景
    ↓
[步骤5] 法律条文检索
  - 检索相关法律条文
  - 检索审查指南
  - 为校对提供依据
    ↓
[步骤6] 整理检索结果
  - 按来源分类整理
  - 标注相关度
  - 保存到 workspace/search-results/
    ↓
输出：结构化检索结果
```

## 检索策略

### 关键词提取规则

1. **技术领域关键词**：从技术方案中提取核心技术领域
2. **技术问题关键词**：提取技术问题相关的关键词
3. **技术方案关键词**：提取技术方案中的关键技术点
4. **同义词扩展**：为每个关键词生成同义词、近义词

### 检索优先级规则

1. **专利检索优先**：优先检索相关专利，了解现有技术
2. **论文检索补充**：检索相关论文，补充技术细节
3. **百科检索辅助**：检索百科知识，补充基础概念

### 相关度评估规则

| 相关度 | 说明 |
|--------|------|
| 高 (0.8-1.0) | 技术领域相同，技术问题相似 |
| 中 (0.5-0.8) | 技术领域相关，有一定参考价值 |
| 低 (0.3-0.5) | 技术领域不同，但有部分参考价值 |
| 不相关 (0-0.3) | 与当前技术方案无关 |

## 输出格式

检索结果保存为JSON格式，文件名格式：`{关键词}-{日期}.json`

保存位置：`workspace/search-results/`
